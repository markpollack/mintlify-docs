<Warning>
**Several agent-engineering projects have moved.** Claude Agent SDK, Agent Client, Agent Judge, Agent Sandbox, and Agent Bench are now maintained at [lab.pollack.ai](https://lab.pollack.ai). They were general-purpose agent libraries that did not depend on Spring AI — see the [migration page](https://lab.pollack.ai/docs/migration/spring-ai-community-to-markpollack) for details. The ACP Java SDK and all other projects listed below remain here.
</Warning>
